# Spotify-Web-App

This is a web application that connects your Spotify account to display your top tracks and artists over short and long time periods in a neat dashboard. Along with the dashboard, there is a community blog page for users to post listening insights and thoughts. This is a portfolio project to practice API integration, HTTP requests, and databases. Currently the web app is in development mode on the Spotify developer dashboard, so API quota is limited to 5 betatesters that can connect their spotify. This web app is deployed on Railway and can be visited here (https://spotify-web-app-production.up.railway.app/)
